Formation Bac +5

Après les 3 ans de BUT informatique vous pouvez poursuivre vos études, même si le principe du BUT est la préparation à l'insertion professionnelle, vous aurez donc plusieurs choix les plus communs qui sont :

Le master

Le master est un cursus de 2 ans après généralement une licence et éventuellement avant un doctorat. Le BUT informatique permet après les 3 ans d’accéder à un master comme ceux ci-dessous :

  • robotique
  • informatique
  • cybersécurité
  • MIAGE
  • ...

Les écoles d'Ingénieurs

Les écoles d’ingénieurs sontune formation de 3 ans qui se fait généralement à la suite d’une classe préparatoire mais il est possible d’en rejoindre une dans le domaine de l’informatique après les 3 ou 2 ans de BUT. Voici quelques une possibles de rejoindre :

  • INSA
  • ENSEEIHT
  • Polytech Montpelier
  • ...
